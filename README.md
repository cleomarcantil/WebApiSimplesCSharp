# WebApiSimplesCSharp

Projeto simples de API em dotNet/C# com funcionalidades básicas implementadas:
  - Autenticação via JWT.
  - Validação de permissões com base em policies.
  - Associação de usuários a várias roles.
  - Consulta via GraphQL
